Hardest Thing I've Learned

The hardest thing I've learned in class so far is the concept of mapping in javascript. All of the work I've done before this class has been in java, so my first instincts for transforming data was to use private methods containing for loops. The idea of having a function as a parameter was confusing at first, but once I got away from object-oriented principles it clicked. The map function handles all of the transformation, so all you have to deal with is writing a function that takes in the current data type and returns the data that you're interested in. It's also important to store the resulting array in its own variable, as you cannot assume that the original array is passed by reference. That was something that tripped me up. Once you understand those concepts, using the map function is actually quite straight-forward.
